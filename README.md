# Cloud-Conference
## 규칙
매주 목요일에 멘토가 주제를 업로드 한다.
- 다음주 수요일까지 주제에 대해서 md파일 작성
- 수요일날 10교시에 주제에 대해 발표할 수 있도록 한다.
- 조사한 파일은 pr 을 날려 컨퍼런스 관리자(가성)의 comment 나 approve 를 받고 merge 를 하는 식으로 진행합니다.
